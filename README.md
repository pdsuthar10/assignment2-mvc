# Rockstar-MVC

## This is an assignment for class CSYE7220 which includes a .NET MVC Project.

### To clone the repository:

```
git clone https://github.com/pdsuthar10/assignment2-mvc.git
```

### Go to src folder
```
cd assignment2-mvc
```

### How to run
```
dotnet run
```

### The webapp will run on http://localhost:5002/
